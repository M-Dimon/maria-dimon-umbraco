Searcable dropdown

Based on the code found on W3 school.
Here it works like a dropdown menu element, and can thus also be used to navigate the site. It is built up around a button that switches between making the dropdown content visible or invisible.

Once the button is pressed, the content and search bar become visible and usable, making it so the user can search for content in the dropdown, making it easier to find and navigate the site.

Currently, I do not know enough about API in regards to Dropdowns, as such I have therefore chosen to omit it in the dropdowns. However, in the future, connecting the dropdown to a database would make it easy to add or remove elements from the dropdown list.

This could be done either by accessing the database directly to enter the information or by incorporating the add or delete as buttons, using JS functions PHP to access the database, thus making it so users themself can edit the dropdown from the site directly.

Option Dropdown

Based on the code found on Educative.io.
Used as a pick one option dropdown. In this case with some video games. It is built around the select element, which activates the dropdown showcasing the list. Unlike the other dropdown, here the search bar is visible and accessible from the start, they do, however, have the same filter function.

Like the other, connecting the dropdown to a database would make it possible to add or remove elements on the list.
